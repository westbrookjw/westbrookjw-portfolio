Projects

A collection of projects that demonstrate my experience with cloud services, IT infrastructure, and practical problem solving.

Secure Static Website Hosting Using AWS Services

This project walks through the process of deploying a secure, high-performance static website using AWS services. It starts with hosting the site on Amazon S3 and configuring CloudFront as a content delivery network (CND) with Origin Access Control (OAC) to securely serve files. It also includes optional steps to set up a custom domain with HTTPS using AWS Certificate Manager and DNS providers like Cloudflare. This setup ensures your website is scalable, secure, and accessible via a personalized domain.
