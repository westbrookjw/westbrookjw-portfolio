Static Website - AWS S3 | CloudFront | Certificate Manager

This project walks through the process of deploying a secure, high-performance static website using AWS services. It starts with hosting the site on Amazon S3 and configuring CloudFront as a content delivery network with Origin Access Control to securely serve files. It also includes optional steps to set up a custom domain with HTTPS using AWS Certificate Manager and DNS providers like Cloudflare. This setup ensures your website is scalable, secure, and accessible via a personalized domain.
