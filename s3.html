S3 Bucket Static Website Deployment

End-to-end cloud-based deployment using AWS and Cloudflare

Project Overview

This project showcases a professional static portfolio website deployed using AWS S3 for hosting, CloudFront for global content delivery and HTTPS, and Cloudflare for DNS management and performance enhancement.

Architecture

  • S3 bucket configured for static website hosting
  • CloudFront distribution pointing to the S3 static site endpoint
  • SSL/TLS enabled via ACM and integrated with CloudFront
  • Cloudflare used for DNS and global CDN routing

Setup Steps

  • Enabled static website hosting
  • Uploaded HTML/CSS/JS/Etc.
  • Created bucket policy - Access only allowed to CloudFront
  • Synced with GitHub actions to update S3 bucket when I push an update to my repository
  • Setup CloudFront to add encryption and for the benefits of a CDN
