S3 Bucket Static Website Deployment

End-to-end cloud-based deployment using AWS and Cloudflare

Project Overview

This project showcases a professional static portfolio website deployed using AWS S3 for hosting, CloudFront for global content delivery and HTTPS, and Cloudflare for DNS management and performance enhancement.

Architecture

  • S3 bucket configured for static website hosting
  • CloudFront distribution pointing to the S3 static site endpoint
  • SSL/TLS enabled via ACM and integrated with CloudFront
  • Cloudflare used for DNS and global CDN routing

Core Features

  • Responsive HTML5 design (Solid State by HTML5 UP)
  • Custom domain managed via Cloudflare
  • HTTPS everywhere with CloudFront and ACM
  • Efficient cache management with CloudFront invalidations
  • Simple deployment via GitHub Actions

Deployment Workflow

This project uses a modern CI/CD pipeline to automate deployment and cache management.

  • Source code managed in GitHub repository
  • On every git push, GitHub Actions triggers a deployment job
  • Updated files are synced to the S3 bucket via aws s3 sync
  • A CloudFront invalidation is automatically created to refresh cached content
  • All secrets and credentials are stored securely in GitHub Secrets
