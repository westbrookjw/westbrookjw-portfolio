S3 Bucket Static Website Deployment

End-to-end cloud-based deployment using AWS and Cloudflare

Project Overview

This project showcases a professional static portfolio website deployed using AWS S3 for hosting, CloudFront for global content delivery and HTTPS, and Cloudflare for DNS management and performance enhancement.

Architecture

  • S3 bucket configured for static website hosting
  • CloudFront distribution pointing to the S3 static site endpoint
  • SSL/TLS enabled via ACM and integrated with CloudFront
  • Cloudflare used for DNS and global CDN routing

S3 Bucket Setup

How to setup an S3 Bucket

  1. Create AWS account.
  2. View all services. Select S3. Create bucket.
  3. Name bucket and leave the rest as is.
  4. Go to Properties. Enable Static website hosting. Specifiy the home page of the website. Leave the rest as is.

CloudFront Setup

How to setup CloudFront

  1. Go to CloudFront then Distributions. Select Create distribution.
  2. Name distribution as you would like and then add your Domain if you have one. If not wait until after next step.
  3. Select "Browse S3" and then select your bucket. Press "Next".
  4. Go to "Origins" then select your "Origin" and press "Edit".
  5. Leave all as is and click "Copy policy".
  6. Go back to "S3", then "your bucket", then "Permissios" and apply this policy to your "Bucket policy".
